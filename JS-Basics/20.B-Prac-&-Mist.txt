Best Practices and Mistakes:
----------------------------





Best Practices:
---------------

1. Avoid Global Variable
2. Avoid new keyword
3. Avoid (==) to compare, use (===) for comparing
4. Always Declare Local Variables
5. Declarations Variable on Top
    let firstName, lastName, price, discount, fullPrice;
6. Declare Objects with const, array
7. Don't Use new Object()
    Use "" instead of new String()
    Use 0 instead of new Number()
    Use false instead of new Boolean()
    Use {} instead of new Object()
    Use [] instead of new Array()
    Use /()/ instead of new RegExp()
    Use function (){} instead of new Function()
8. Beware of Automatic Type Conversions
9. Use Parameter Defaults in Function
    function(a=1, b=2){};
10. End Your Switches with Defaults -> loop






JavaScript Common Mistakes:
---------------------------

1. Using the Assignment Operator (=) instead of (==) in comparing
2. Using (==) to compare, instead of Using (===) in comparing
3. Confusing Addition & Concatenation (+)
4. Not using (;) after writing a statement. Never break a return statement.
5. Undefined is not null
